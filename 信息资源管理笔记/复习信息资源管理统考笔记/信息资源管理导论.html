信息资源管理概论

1.社会信息化包括哪几个层次

三个层次

  1. 生产工具信息化
  2. 社会生产力信息化
  3. 社会生活信息化。

2.什么是信息资源

人类社会信息活动中围绕信息的搜集，整理和利用而开展的一系列社会经济活动。

3.信息资源有哪些特征

  1. 人类需求性
  2. 稀缺性
  3. 可选择性
  4. 共享性
  5. 时效性
  6. 不可分割性
  7. 不同一性
  8. 驾驭性

4.信息资源有什么功能

  1. 经济功能
  2. 管理与协调功能
  3. 选择与决策功能
  4. 研究与开发功能

5.试述信息资源管理的发展过程

  1. 传统管理阶段
  2. 信息管理阶段
  3. 信息资源管理阶段

6.信息资源管理的目标是什么

保证信息资源的开发利用在有领导、有组织的统一规划和管理下，协调一致、有条不紊地进行，使各类信息资源以更高的效率、效能和更低的成本在国家社会进步、经济发展、人民物质文化生活水平的提高中充分发挥应有的作用

7.信息资源管理的任务包括哪些方面

  1. 制定信息资源的开发战略、规划、方针和政策。
  2. 制定信息资源管理的法律、规章和条例。
  3. 利用法律和行政手段协调各部门、各地区和各企业之间的关系。
  4. 加强国家信息基础设施和信息资源管理网络的建设。

8.信息资源管理有何意义

  1. 信息资源管理开辟了管理新天地
  2. 合理开发和有效利用的必要条件，保证其在成本最低时经济功能最强
  3. 有利于保证信息资源开发利用机构的合法权益

9.宏观层次的信息资源管理应遵循哪些原则

  1. 信息资源是一种重要的经济资源
  2. 实行分级分类管理
  3. 国家的信息资源管理，主要是确定目标、进行投资决策，并为各级政府业务部门中观层次的信息资源管理提供条件
  4. 大力推广使用现代信息技术，提高信息资源的开发水平和利用效果
  5. 确定信息资源管理的保密和保存制度
  6. 协调与其它国家之间的信息资源交流关系

10.信息资源管理有哪些手段

  1. 技术手段
  2. 经济手段
  3. 法律手段
  4. 行政手段

11.社会信息化

人类对信息的依赖程度越来越高，而对物质和依赖程度相对降低。

12.经济信息化

主要指信息要素渗透到经济活动之中，社会经济的发展主要不是依赖物质材料的增加和新能源的开发，而是依赖信息力量的推动。

13.信息资源

信息资源是指人类社会信息活动中积累起来的以信息为核心的，围绕信息的搜集、整理、提供和利用而开展的一系列社会经济活动。

14.记录型信息资源

包括传统介质（纸张竹帛等）和各种现代介质（磁盘、光盘等）记录和存储的知识信息。它是信息资源存在的基本形式，也是信息资源的主体。

15.零次信息资源

指各种渠道中由人的口头传播的信息。比如书信、论文手稿、笔记、实验记录、会议记录等

16.智力型信息资源

这类信息资源主要表现为人脑存贮的知识信息，包括人们掌握的诀窍、技能和经验，又称为隐性知识。

17.简述信息资源的选择与决策功能

  1. 没有信息就无任何选择和决策可言
  2. 没有信息的反馈，选择和决策就无优化可言

18.信息资源管理的基本要素

  1. 管理对象：信息资源
  2. 实施领域：国际、国家和社会组织
  3. 实施方法：综合使用各种管理手段，包括技术手段、经济手段、人文手段等
  4. 信息资源管理的实质，即一种管理思想、管理模式和管理活动

19.简述信息资源管理概念提出的两个背景

  1. 信息管理阶段纯粹的技术手段不能实现对信息的有效控制和利用
  2. 当代社会经济发展使得信息成为一种重要的资源，迫切需要从经济的角度思考问题，并对这种资源进行优化配置和管理

信息资源的优化配置

1.什么是信息资源配置

指为最大限度减少宏观经济浪费和现实社会福利最大话而对现代技术成果与各种投入要素进行的有机组合。

2.信息资源配置应遵循哪些原则

  1. 社会经济福利最大化原则
  2. 需求导向原则
  3. 公平原则
  4. 市场手段和政府手段互补原则

3.试述信息资源配置的主要内容

  1. 横向配置，部门、行业或地区地宏观布局，包括区域配置和行业配置。
  2. 纵向配置，只有符合资源增值地投资项目，才能吸引资源地流入。

4.网络环境下的信息资源应如何配置

  1. 时间矢量配置
  2. 空间矢量配置
  3. 品种类型配置

5.信息资源配置的机制是什么

  1. 市场配置
  2. 政府配置
  3. 产权配置

6.试述信息资源配置市场失灵的原因

  1. 信息的外部效应
  2. 信息的公共物品的属性
  3. 信息商品的垄断性
  4. 信息市场的信息不对称
  5. 信息活动的非营利性

7.政府在信息资源配置方面有何作用

政府配置机制是指政府利用政策、法律、税收工具，或者通过直接投资和财政补贴来调整信息产出。

  1. 信息基础设施建设
  2. 公共信息服务
  3. 教育
  4. 科学研究
  5. 制度建设

8.信息资源产权配置的功能主要体现在哪些方面

  1. 可以减少浪费，提高经济效率
  2. 稳定的产权格局或结构，会形成一种资源配置的客观状态
  3. 产权的变动同时也改变资源配置格局

9.应从哪些方面考察信息资源配置的有效性

  1. 信息生产的有效性
  2. 信息商品的生产比例的有效性
  3. 信息市场与交换的有效性

10.试述我国文献信息资源的共享模式

  1. 垂直型（纵向）共享
  2. 水平型（横向）共享
  3. 网络型共享

11.信息资源配置的经济意义

  1. 有效配置信息资源有利于更好地满足人类对资源地需求
  2. 有效配置信息资源有利于最大范围内实现资源共享
  3. 有效配置信息资源有利于防止信息资源污染，实现社会可持续发展

12.信息资源的纵向配置

指的是只有符合资源增值的投资项目，才能吸引资源的流入。

13.信息市场失灵

信息市场的自组织机制，对信息资源的市场配置存在着一些自身无法克服的缺陷，即市场失灵

14.信息资源的政府配置

指政府利用政策、法律、税收工具，或通过直接投资和财政补贴来调整信息产出

15.信息资源产权配置的功能

通过调整和明晰产权，优化信息资源配置

16.如何衡量信息资源配置的效率

  1. 总体社会资源优化配置和社会福利最大化的宏观目标
  2. 必须立足于信息生产、信息服务有效的微观基础
  3. 衡量信息资源配置效率需要从宏观，中观和微观三种不同的层次来加以考察

17.网络信息资源的空间矢量配置

网络信息资源不在同地区、不同部门之间分布，实质上是不同使用方向上的分配

及时性、准确性、综合性、获取简易性、经济性等

  1. 能够及时以及适当的方式提供解决问题所需要的依据

  3. 信息的可信赖度高
  4. 信息具有综合性
  5. 信息容易获取
  6. 信息费用与目标吻合
  7. 信息发生源
  8. 信息载体
  9. 意图
  10. 可靠性
  11. 从不同的信息源获得同一性质的信息
  12. 内容
  13. 原因
  14. 时间
  15. 地点
  16. 人
  17. 方法
  18. 途径
  19. 状况
  20. 存在率的评价
  21. 适合率的评价
  22. 可靠性的评价

2.企业信息资源管理的任务是什么

企业为达到预期目标，运用现代的管理方法和手段对与企业相关的信息资源和信息活动进行组织，规划，协调和控制，以实现对企业信息资源的合理开发和有效利用。

企业信息资源管理的任务就是要有效地搜集、获取和处理企业内外信息，最大限度地提高企业信息资源的质量、可用性和价值，并使企业各部门能够共享这些信息资源

3.企业信息资源管理的主要内容是什么

  1. 提高企业全体人员对信息价值的认识
  2. 促进信息在企业的共享
  3. 为企业建立合理的信息结构
  4. 保证信息的相关性和标准化
  5. 提高信息质量，提高信息安全
  6. 保证信息资源管理系统的可用性
  7. 促进企业员工利用信息

4.试述企业信息资源管理的技术框架

  1. MIS 管理信息系统
  2. Intranet 及其应用
  3. Extranet

5.试述企业信息资源管理的组织结构

  1. 企业信息总监 CIO
  2. 信息技术人员
  3. 信息管理人员
  4. 信息资源管理辅助人员

6.什么是知识管理

知识管理是对知识及其创造，收集，组织，传布，利用与宣传等相关过程的系统管理

7.企业知识管理的目标是什么

  1. 提高企业创新能力
  2. 提高企业反映能力
  3. 提高企业效率
  4. 提高企业员工技能
  5. 实现企业知识资产价值

8.试述企业知识管理的职能

  1. 为联机和非联机知识建立知识地图
  2. 向用户提供知识利用方面的指导和培训，并提供相应的知识检索工具
  3. 监视企业外部知识源

具体来说：

  1. 外化
  2. 内化
  3. 中介
  4. 认知
  5. 评测

9.企业知识管理的基本流程包括哪些步骤

  1. 制定知识管理战略
  2. 确定知识管理重点领域
  3. 评估企业知识资源
  4. 制定知识管理方案
  5. 实施知识管理方案
  6. 监督知识利用

10.企业知识管理的实现技术主要

  1. 网络技术
  2. 群件技术
  3. 知识库技术
  4. 其它重要技术，Push技术、决策支持系统与专家系统技术、信息检索技术、超文本与电子出版技术

11.知识管理

是对知识及其创造、收集、组织、传播、利用与宣传等相关过程的系统管理。

12.记录管理

记录管理又称文书管理，记录管理政府内部业务工作的信息

13.简述企业信息的特点

  1. 信息的渠道来看，企业不但要搜集、存储和利用外部信息，还要对企业内部信息进行搜集、加工和存储。
  2. 从信息载体来看，既有传统的印刷型，又有磁盘、光盘、微缩胶片、胶卷、网络等新型存储介质，既有文字型，又有实物型；既有书本型，又有报表式；既有正式文献，又有非正式书信。
  3. 从信息内容来看，既有科技信息，又有经济信息、财务信息、市场信息等
  4. 从时效性看，企业信息更要求新颖及时

14.企业信息资源管理的主要内容是什么

  1. 提高企业全体人员对信息价值的认识并促进企业活动对信息的需求
  2. 促进信息在企业不同部门和不同群体之间的共享
  3. 为企业建立合理的信息结构
  4. 保证信息的相关性和标准化，以及最大限度地减少信息重复现象
  5. 提高现象地质量，保证信息地安全
  6. 保证信息资源管理系统地可用性和充分运行，以便在任何时候和地点均可及时检索到所需要地信息
  7. 促进企业员工利用信息

1.试述政府信息资源管理产生的背景

  1. 信息资源观的转变
  2. 信息技术的发展
  3. 经济信息化的推进
  4. 政府体制改革的需求

2.试述政府信息资源管理的发展历程

  1. 记录管理：文书管理
  2. 信息技术管理
  3. 政府信息资源管理
  4. 电子政务

3.政府信息资源管理有哪几种类型

  1. 可以完全对社会公开的信息
  2. 只在部门之间共享的信息
  3. 只在内部共享的信息
  4. 只对特定的个体开放的信息

4.政府信息资源管理有哪些特点

政府信息具有数量大、价值高、储存分散、搜寻难等特点

5.试分析政府信息资源共享和保密的关系

公开是为了让公众共享政府信息和接受公众的舆论监督 保密则是为确保特定对象对政府信息的有效共享和对政府行为的及时监督。

6.政府信息资源管理有何意义

  1. 政府信息资源管理为实现政府信息化保驾护航
  2. 政府信息资源管理与国家安全息息相关
  3. 政府信息资源管理为培养优秀的政务人员创造条件

7.政府信息资源管理应遵循哪些原则

  1. 领导必须重视信息工作
  2. 制定政府信息资源开发和利用规划
  3. 建立有效的政府信息资源网络
  4. 制定政府信息资源管理政策

8.政府信息资源管理的主要任务是什么

  1. 增强信息意识
  2. 提高政府信息资源开发和利用的能力
  3. 最大限度地降低政府信息活动的费用
  4. 保障国家信息安全

9.试述政府信息资源管理的主要内容

  1. 政府信息化政策的制定
  2. 政府信息化发展计划
  3. 政府信息化的项目管理
  4. 政府信息的管理
  5. 政府信息基础设施的管理
  6. 政府信息技术的管理
  7. 政府应用信息系统管理
  8. 政府信息化的人力资源开发
  9. 政府信息资源管理相关的法律和法规
  10. 政府信息的安全管理

10.试述政府信息资源建设的内容

  1. 电子政务：G2G，G2B，G2C
  2. 电子政务的系统构成：网络应用平台子系统，网络安全管理子系统，信息发布子系统，公文运转子系统，经济计划管理子系统，项目管理子系统
  3. 我国电子政务发展对策分析：首先要转变观念，充分认识发展电子政务的重要性和紧迫性，从战略高度重视电子政务的发展

11.试述政府信息化与电子政务的关系

政府信息化是指用信息技术全面推进政府部门的办公自动化，利用信息技术、信息资源来提高政府的决策水平，和公务人员的信息能力的过程

电子政务，利用计算机和通信技术，建立的网络化的政府信息系统，并为企业，公众提供政府信息服务

12.政府信息化建设的优势与劣势分别是什么

优势：

  1. 从动机来看，政府部门是进行信息化建设的倡导者
  2. 从资金来看，政府部门解决了信息化建设所需的财政资金问题
  3. 从信息技术的应用来看，政府部门是现代信息技术的推进者

劣势：

  1. 政府部门习惯的管理模式和思维上的惰性是信息化建设的首要障碍
  2. 对信息技术的应用认识不当
  3. 政府部门信息建设各自为政的作风阻碍信息的共享

13. 我国政府信息化建设

  1. 起步阶段（20 世纪 80 年代初 —— 20 世纪 90 年代初）
  2. 推进阶段（20 世纪 90 年代初 —— 20 世纪 90 年代末）
  3. 发展阶段（1999 —— 2001年）
  4. 高速发展阶段（2002 至今）

14.电子政务

电子政务，利用计算机和通信技术，建立的网络化的政府信息系统，并为企业，公众提供政府信息服务

15.G2B电子政务

G2B是指政府通过电子网络进行电子采购与招标，为企业提供各种服务

信息政策与法律

1.信息政策包括哪些要素

  1. 法规问题
  2. 宏观经济问题
  3. 组织问题
  4. 社会问题

2.试述信息政策的体系结构

三个层次：

  1. 产业政策层面
  2. 组织政策层面
  3. 社会政策层面

五个因素：

  1. 信息技术因素
  2. 信息市场因素
  3. 信息管理因素
  4. 人力资源因素
  5. 信息法规因素

3.信息法律的主体和客体分别指什么

信息法律的主体，信息法律关系中依法享有权力承担义务的人或组织。

信息法律的客体，物化的和非物化的财产

4.信息法律与信息政策的区别是什么

  1. 信息法律是一种法律手段；而信息政策是一种行政手段
  2. 信息法律的问题、对象比信息政策更基本，更普遍、时效更长、相对更稳定；信息政策对象范围和有效时间范围比信息法律要小，更带有阶段性和灵活性
  3. 信息政策是信息立法的基础

5.构建信息法律体系遵循什么原则

  1. 创建适于各行的政策法律环境
  2. 必须考虑到信息技术的未来发展
  3. 考虑到信息活动的跨行业、跨地域的特性

10.帕累托最优

无论作何改变都不可能同时使一部分人受益而其他人不受损，也就是说经济运行达到有效时，一部分人进一步改善处境必须以另一些人处境恶化为代价。经济学家经常将资源地最大效率配置称为“帕累托最优”

  1. 市场可以减少信息产生地不确定影响
  2. 市场能自动的使企业、个人为信息生产提供动力
  3. 市场通过价格信号引导信息产生
  4. 市场竞争的压力迫使企业不断创新
  5. 市场制度有助于培育创新的企业家
