档案学基础

1.试述档案学的研究对象与范围 P413

研究对象：

  1. 档案
  2. 档案工作
  3. 档案事业
  4. 档案学科本身。

研究范围：

  1. 档案学基础理论研究
  2. 档案学应用理论研究
  3. 档案学应用技术研究
  4. 档案学相关的内容

2.试述欧美档案学主要研究内容与特点 P417

  1. 西欧档案学理论具有较扎实的基础
  2. 北美是借鉴吸收欧洲档案学理论的基础上，结合北美国家档案管理的传统来研究文件管理理论和档案管理理论。

3.欧美档案学对我国档案学研究具有哪些借鉴意义

  1. 中国古代档案工作的整体发展水平领先于西方，但中国档案学产生晚于西方
  2. 中国档案学的研究对象局限于档案现象，而西方将档案学的研究对象扩大至广义的文件现象
  3. 中国档案学以马克思主义哲学为指导思想，侧重于历史方法和逻辑方法
  4. 西方档案学以不同的哲学思想为指导，运用多门学科的研究方法
  5. 中国许多档案学者或缺良好的高等教育，或缺丰富的实践经验
  6. 西方档案学者大多两者兼具

4.简述我国档案学发展阶段及其主要特点 P422

  1. 形成时期（1949年10月——1965年）：档案学作为一门独立学科为国家所确立
  2. 初步发展时期（1957——1966年）：开创性、针对性等特点
  3. 停滞时期（1966——1967年）：在这一时期没有任何发展
  4. 恢复时期（1976——1982年）：档案学的学科地位被重新恢复和确立
  5. 全面发展时期（1983——）：档案学专业从历史文献学中独立，档案学从历史学科规划到管理学，我国档案事业进入法制轨道，形成完整的教育体系

5.为什么说档案具有凭证价值 P427

  1. 档案凭证价值，是档案不同于和优于其他各种资料的最基本的特点
  2. 档案凭证价值，是由档案形成过程及其结果的内容特点决定的
  3. 它客观地记录了以往的历史情况，是历史证据

6.实现档案价值具有哪些规律性 P428

  1. 档案价值的扩展律
  2. 档案价值的时效律
  3. 档案价值的条件律

7.我国档案工作的主要内容有哪些 P430

  1. 档案管理工作
  2. 档案行政管理工作
  3. 档案教育工作
  4. 档案宣传工作
  5. 档案科学技术研究工作
  6. 档案国际合作交流工作

8.我国档案工作的基本原则是什么 P432

  1. 全国档案工作实行统一领导、分级管理
  2. 维护档案的完整与安全
  3. 便于社会各方面的利用，满足社会对档案的需要

9.档案事业行政管理机构的主要职责是什么 P436

在统一管理党、政档案工作的原则下，分层负责地掌管国家档案事务

10.档案室的主要职责是哪些 P436

  1. 管理本机关的档案
  2. 为党和国家积累档案史料
  3. 对本机关文件材料的归档作指导和监督
  4. 管理本单位的档案和相关资料
  5. 组织，利用档案和相关资料
  6. 向档案馆移交有长远保存价值的档案

11.我国档案馆有哪些类型 P436

  1. 国家档案管
  2. 专业档案馆
  3. 企业档案馆

12.试述档案事业管理体制 P438

指档案事业管理的体系和组织制度，包括档案行政管理部门的设置及其隶属关系、权限划分等

档案事业管理体制类型：分散式，集中式

13.我国档案法规体系由哪些部分构成 P438

  1. 档案工作法律
  2. 中央档案行政法规
  3. 中央档案行政规章
  4. 地方性档案工作法规
  5. 地方性档案工作行政规章

14.《档案法》的主要内容是什么 P439

共六章27条。其中包括：

  1. 总则
  2. 档案机构及其职责
  3. 档案的管理
  4. 档案的利用和公布
  5. 法律责任和附则

15.档案工作职业道德包括哪些内容？如何理解 P441

忠于职守，爱岗敬业，遵纪守法，严守机密，博学求进，公正服务

16.应用档案学 (P416)

研究档案管理中的原则、问题、方法与技术的应用。

其分支学科是：档案事业管理学、档案管理学科技档案管理学、专门档案管理学、档案文献编纂学、档案法规学等

17.档案的本质属性 (P423)

原始记录性是其本质属性

档案是人们在社会实践中直接形成的原始性信息记录，对以往社会实践具有直接的原始记录作用

18.公务档案 (P426)

公务档案指人们在公务活动中形成的档案

公务文书，法律、法规、行政公文等

19.现行档案 (P426)

指形成时间较晚，离现在的时间较近，且主要起现时性查考作用的档案，在我国一般指1949年新中国成立之后的档案

20.科技档案 (P426)

指人们在科技、生产活动中形成的，由科技文件转化而成的档案，如图纸、设计任务书、科研报告等

21.档案馆 (P426)

指永久保存档案的基地，利用档案信息的文化事业机构，属于国家文献信息系统的重要组成部分

22.专门档案 (P426)

是通过选择并保存下来，以备查考的，各种专门文件组成的有机体系的总称。 如会计档案、人事档案、诉讼档案、病例档案

23.人事档案 (P427)

人事档案是一种专门档案。 是记录一个人的个人情况的文件材料，起着凭证、依据和参考的作用

24.革命历史档案 (P427)

革命历史档案，又称革命政权档案，指1949年10月1日中华人民共和国成立之前，由中国共产党及其所领导的军队、政权、企事业单位、社团等社会组织及个人所形成的归国家所有的档案

25.档案价值 (P427)

档案的价值，是指档案对国家、社会组织或个人的有益性、有用性

26.档案价值形态 (P427)

是指档案价值的具体表现形式

27.档案价值时效 (P428)

档案的有用性是有时限的，某些档案在一定时期内对利用者是有价值的，超过这个时间限制后则降低或丧失了价值

28.档案的行政作用 (P429)

档案对于各级各类机构、社会组织，国家工作人员的，政策、体制、秩序、决策的科学性具有【凭证】和【参考作用】，这种作用可以称为资质作用或行政作用

29.档案的法律作用 (P429)

指档案在解决争端、处理案件等活动中所发挥的证据作用

法律作用是档案凭证价值的集中体现

30.档案工作标准 (P434)

  1. 是档案工作中为事物和概念制定的各种工作标准的总称
  2. 是档案工作中单位和个人应当遵守的共同准则和依据

31.档案室 (P436)

是机关组织，统一保存和管理档案的内部机构，是整个机关的组成部分

32.中国第一历史档案馆 (P437)

中国第一历史档案馆，其前身是北京故宫博物馆文献部，1951年改称档案馆，1955年故宫档案馆划归国家档案局领导，改称现名。该馆是以管理明清时期中央机关档案馆为主的文化事业机构，馆藏明清两代重要机关和少数地方机关的档案共74个全宗，1000多万件

33.文件中心 (P438)

一种社会化、集约化和专业化的档案文件管理机构，介于文件形成单位和档案馆之间的一种过渡性档案文件管理机构

34.档案事业管理体制 (P438)

指档案事业管理的体系和组织制度，包括档案行政管理部门的设置及其隶属关系、权限划分等

35.档案学的学科特点 (P414)

  1. 综合性
  2. 应用性
  3. 实践性

36.档案的文化性 (P424)

档案是人类社会发展到一定历史阶段的产物，是人类文明进化的结果。 档案是人类社会连续性发展的桥梁，是人类社会历史的浓缩，透过档案我们能够触摸到人类社会绵延不断的“脉搏”

37.简述档案信息化建设的内容 (P435)

  1. 档案信息数字化
  2. 档案网站建设
  3. 数字档案馆建设

38.档案室的任务是什么？(P436)

档案室机关统一保存和管理档案的内部机构，是整个机关的组成部分

  1. 管理本机关的档案
  2. 为党和国家积累档案史料
  3. 对本机关文件材料的归档作指导和监督
  4. 管理本单位的档案和相关资料
  5. 组织，利用档案和相关资料
  6. 向档案馆移交有长远保存价值的档案

文件管理

1.简述文书与文件的基本概念 P443

文书：它所包含的对象是公务文书和私人文书，即公务文书和私务文书的总称

文件：含义为领导机关所制发的，具有法定效力，并设有特定版头的公文

2.公文概念是什么？他有哪些特点 P444

公文是法定机关与组织在公务活动中，形成的书面文字材料，作为传达意图、办理公文与记载活动工作的一种工具。

公文的特点：

  1. 是由法定作者制成和发布的
  2. 具有法定的权威和效力
  3. 具有规范的体式
  4. 具有特定的处理程序
  5. 具有现行效用

3.试述公文的作用 P445

  1. 法规作用
  2. 书面指导作用
  3. 公务联系作用
  4. 宣传教育作用
  5. 凭证和依据作用

4.简述专用文件和通用文件的概念及种类 P447

通用文件：单位在工作活动中普遍使用的文件。如命令、决定、指示、请示、批复、报告、通知等

专用文件：是指在一定范围，根据特殊需要专门使用的文件，如外交文件、军事文件、司法文件等

5.公文的体式及撰写要求是什么 P449

文理通顺、便于阅读、准确无误、讲求实效、严谨周密、简明精炼、庄重得体、符合规范、整齐划一等

6.简述文书工作的组织形式 P451

文书工作是指在社会组织内部围绕着文书工作的活动程序。

组织形式：集中式、分散式、复合式

7.什么叫行文规则？机关之间的行文方式有哪几种 P451

行文规则，各社会之间文件往来运行所必须遵守的统一规则。

机关之间的行文方式主要有下行文、上行文和平行文三种。

8.简述文书处理的程序 P452

  1. 收文处理程序：文件的收进、启封、登记、分类、分送、拟办、承办、催办、注办、清退、归档等
  2. 发文处理程序：拟稿、核稿、签发、缮印、校对、盖印、登记、封装、发出、注办、归档等

9.归档文件的整理原则与质量要求是什么 P455

  1. 归档文件的整理原则：即遵循文件的形成规律，保持文件之间的有机联系，区分不同价值，便于保管和利用
  2. 归档文件的质量要求：归档文件应齐全完整，整理归档文件所使用的书写材料、纸张、装订材料等应符合档案保护要求

10.归档文件的整理方法包括哪几个方面 P456

  1. 装订
  2. 分类
  3. 排列
  4. 编号
  5. 编目
  6. 装盒

11.试述电子文件的含义和特点 P457 P459

电子文件是以代码形式记录于磁盘、磁带、光盘等载体，依赖计算机系统存取并可在通信网络上传输的文件

电子文件的特点：

  1. 信息的非人工识读性
  2. 系统依赖性，必须借助于计算机
  3. 信息与特定载体之间的可分离性
  4. 信息的易变形
  5. 信息存储的高密度性
  6. 多种信息媒体的集体性
  7. 信息的可操作性

12.电子文件的管理原则有哪些 P460

  1. 全程管理原则
  2. 前端控制原则
  3. 真实性保障原则
  4. 完整性保障原则
  5. 可读性保障原则

13.电子文件的管理有哪些模式 P462

  1. 机构内部电子文件的管理模式
  2. 档案馆电子文件的管理模式
  3. 数字档案馆

14.电子文件鉴定包括哪些内容 P465

  1. 电子文件的鉴别
  2. 电子文件的内容鉴别
  3. 电子文件的技术鉴定
  4. 电子文件的处置

15.简述元数据的含义及与著录信息的区别 P467

元数据是数据的数据。电子文件管理中的元数据是指电子文件系统自动记录的关于文件形成时间、地点、人员、活动、文件系统、结构及内容等方面的具体数据。

元数据与著录信息区别： 1. 就内容而言：两者呈交叉关系，著录信息无法从元数据中提取，需要人工编制而成 2. 就作用而言：元数据的作用要广泛的多，著录的作用包括登记、介绍、报道、交流和检索 3. 就生成方式而言，元数据是文件管理系统在文件形成与管理过程中自动形成的，而著录信息的生成则是借助于自动著录和人工著录相结合的方式

16.电子文件管理系统的实现方式有哪些 P471

  1. 嵌入式
  2. 联合式
  3. 独立式

17.行政文件 (P447)

是指国家机关在日常公务活动中形成和使用的文件，具有行政指挥、领导指导和公务联系的作用

18.归档文件整理 (P455)

指的是将归档文件以件为单位进行装订、分类、排列、编号、编目、装盒、使之有序化的过程

19.电子文件 (P457)

是以代码形式记录于磁带、磁盘、光盘等载体，依赖计算机系统存取并可在通信网络上传输的文件

20.文件生命周期理论 (P480)

是研究文件从形成到销毁或永久保存的整个运动过程，是对文件运动过程和规律的客观描述的科学抽象

21.《档案的整理与编目手册》(P495)

1898年，荷兰三位档案学者萨穆•缪勒、约翰•斐斯和罗伯特•福罗英出版了专著《档案的整理与编目手册》。 三位学者通过科学阐释全宗的定义、性质以及全宗内档案整理系统的特点，完成了对来源原则的理论论证

22.来源原则 (P473)

就是指档案馆按照档案的来源进行整理和分类，要求保持同一来源的档案不可分散、不同来源的档案不得混淆的整理原则

23.公文的特点 (P444)

  1. 是由法定作者制成和发布的
  2. 具有法定的权威和效力
  3. 具有规范的体式
  4. 具有特定的处理程序
  5. 具有现行效用

24.公文的现行效用 (P445)

所谓现行效用，是指公文在其内容所针对的现行公务活动中直接发挥实际效力，具有依据和凭证功能

25.简述文书处理程序 (P452)

文书处理程序指的是在一个机关内部，按照公务文书的制发、办理与管理的规律，对文书工作的一系列操作环节、工作步骤的有序组合和合理安排。

  1. 收文处理程序：文件的收进、启封、登记、分送、拟办、承办、催办、注办、清退
  2. 发文处理程序：拟稿、核稿、签发、缮印、校对、盖印、登记、装封、发出、注办、归档等

26.归档电子文件的质量要求

  1. 归档的光盘应清洁，无病毒、无划痕，归档信息填写清楚
  2. 电子文件归档，要提供电子文件的技术环境、相关软件、版本、数据类型、格式、被操作数据、检测数据等相关信息
  3. 电子文件存储载体，要采用只读光盘
  4. 归档的电子文件必须是可读文件
  5. 特殊格式的电子文件，应在存储载体中同时存有相应的支持软件。

档案管理

1.简述来源原则的基本内容 P477

尊重来源，尊重全宗的完整性，尊重全宗的原始整理体系

2.试述文件生命周期理论的基本内容 P481

文件生命周期理论的基本内容，主要有三点：

  1. 文件从其形成到销毁或永久保存，是一个完整的运动过程
  2. 由于文件价值形态的变化，这一完整过程可划分为若干阶段
  3. 文件在每一阶段因其特定的价值形态而与服务对象、保存场所、管理形式之间存在一种内在的对应关系

3.试述档案鉴定标准 P490

  1. 档案的来源标准
  2. 档案的内容标准
  3. 档案的形式特征标准
  4. 档案的相对价值标准
  5. 档案的效益标准

4.档案收集的概念与内容是什么 P492

档案收集的概念：就是按照档案形成的规律，把分散在各机关、个人手中及其其他地方的档案材料接受、征集、集中起来

档案收集的内容：

  1. 对本机关需要归档案卷的接受工作
  2. 对机关具有长久保存价值的档案的集中和接受工作
  3. 对历史档案的接收和征集工作

5.档案整理工作的内容与原则是什么 P495

档案整理工作的内容：区分全宗、全宗内档案的分类、确立保管单位、案卷的排列和案卷目录的编制

档案整理工作的原则：

  1. 充分利用原有的整理基础
  2. 必须保持文件之间的历史联系
  3. 必须便于保管和利用

8.档案鉴定的内容与方法是什么 P501

档案鉴定的内容：

  1. 制定鉴定档案价值的有关标准
  2. 判定档案材料的价值
  3. 确定其保管期限
  4. 拣出无保存价值和保管期满的档案，进行销毁或作相应的处理

鉴定档案价值的基本工作方法是直接审查档案，通常把这种方法称为直接鉴定法

9.简述档案鉴定工作的概念及保管期限表的类型 P501

档案鉴定工作的概念：指对档案真伪和档案价值的鉴定

  5. 机关档案保管期限表

10.简述档案保管的概念与内容 P502

档案保管的概念：是指根据档案的成分和状况所采取的存放和安全防护措施

档案保管的内容：

  1. 档案的库房管理
  2. 档案流动过程中的保护
  3. 保护档案的专门措施

档案统计，以表册、数字的形式，揭示档案和档案工作有关情况

组织沿革是系统记载一个机关、地区或专业系统的体制、组织机构和人员编制等反面变革情况的参考资料。

  4. 重大的外事活动，劳动模范、先进集体的材料，相关照片，当年的大事记，等等

16.《中华人民共和国档案法》的作用

《中华人民共和国档案法》规定：属于国家所有的档案，由国家授权的档案馆或者有关机关公布；未经档案馆或有关机关同意，任何组织或个人无权公布。集体所有的和个人所有的档案，档案的所有者有权公布，但必须遵守国家有关规定，不得损害国家安全和利益，不得侵犯他人的合法权益，必要时应当申请当地档案行政管理部门的批准。 《中华人民共和国档案法》制定并颁布，说明我国档案事业开始进入法制轨道。

  2. 尊重全宗的完整性：全宗是一个整体，整理档案必须维护全宗的完整性，同一个全宗的档案不可分散，不同全宗的档案不得混淆。
  3. 尊重全宗内的原始整理体系，尊重全宗的原始整理顺序和方法，不轻易打乱重整

来源原则的理论意义和实践价值：

  1. 为档案馆馆藏的实体整理和分类提供了合理的客观依据
  2. 保持了档案的本质属性
  3. 来源原则是兼具理论性和实践性的管理思想和原则。
  4. 确立了档案实体整理和分类的基本单元
  5. 它划清了档案馆与图书馆的界限，保证了档案专业的独立性
  6. 是档案理论研究的原则之一，有助于揭示档案的来龙去脉，保持历史的本来面貌

内容：

  1. “从社会的总体需求出发”的鉴定基本指导思想和出发点
  2. “全面、历史、发展”这三个鉴定基本观点
    1. 全面的观点要求全面分析档案；
    2. 历史的观点要求必须结合档案形成的历史环境和条件来分析其价值
    3. 发展的观点要求尊重档案价值的发展特点和规律
